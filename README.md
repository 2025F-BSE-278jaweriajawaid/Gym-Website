# Gym Management System (Flask)

## Project Overview
This is a basic Gym Management System website developed using Flask, HTML, and CSS.
It includes multiple pages using Flask routes and templates.

## Pages
- Home Page: Welcome message and gym highlights
- About Us Page: Gym mission, team, and facilities
- Membership Page: Membership plans and services

## Technologies Used
- Python (Flask)
- HTML
- CSS

## Folder Structure
Gym-Website/
├── app.py
├── templates/
│   ├── home.html
│   ├── about.html
│   └── membership.html
├── static/
│   └── style.css
└── README.md

## How to Run the Project
1. Install Python
2. Install Flask:
   pip install flask
3. Run the app:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/
